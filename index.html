01
Create the Google Form
Run an Apps Script to generate the response form in your Drive
▾
You need to collect course registrations from your FYAT members. Run this one-time script to auto-create a form in your Google Drive — no manual setup needed.
  • 1
    Go to script.google.com and click New Project
  • 2
    Delete all existing code in the editor
  • 3
    Paste the code below, then press Run (▶)
  • 4
    Grant permissions when prompted — Google will ask to access your Drive
  • 5
    A new form titled "FYAT Session Availability" will appear in your Google Drive
  • 6
    Share the form link with all FYAT members and ask them to fill it out
apps-script.gs
function createFYATForm() { var form = FormApp.create('FYAT Session Availability'); form.setCollectEmail(true); form.addTextItem() .setTitle('Name') .setRequired(true); form.addTextItem() .setTitle('Student ID') .setRequired(true); form.addParagraphTextItem() .setTitle('Registered Courses. Format: [Course1-Section1, Course2-Section2,...]') .setHelpText('Enter your courses like: CSE110-02, MAT110-05, ENG101-11. ' + 'Use the exact course code and section number from your registered routine.') .setRequired(true); var url = form.getPublishedUrl(); Logger.log('Form created: ' + url); SpreadsheetApp.getUi(); }
02
Download Responses as CSV
Export form responses from Google Sheets
▾
Wait until all members have submitted. You can re-run anytime with an updated CSV — just go through Steps 3–4 again.
  • 1
    Open your form in Google Forms and click the Responses tab at the top
  • 2
    Click the Google Sheets icon (spreadsheet icon) to open responses in Sheets. If not linked, click the link icon to create one
  • 3
    In the Sheets file, go to File → Download → Comma Separated Values (.csv)
  • 4
    Save the CSV somewhere easy to find — you'll upload it in the next step
The CSV should have columns in this order: Timestamp, Email, Name, Student ID, Courses. This is the default Google Forms export format — no editing needed.
